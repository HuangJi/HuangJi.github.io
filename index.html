Fleat

Invited to Stanford TreeHacks. We built a system by which people can manipulate drone by making gestures in 2 days.

史丹佛大學 TreeHacks 主辦的三天兩夜的國際黑客松，受理報名件數超過 1300 件，2015HackNTU 核心團隊受邀參加，現場有許多來自世界各地的開發者和矽谷科技公司，齊聚一堂做出能夠 Live Demo 的產品來展示。而我們的作品名稱叫做 Fleat，是利用主辦方現場提供的硬體設備，Kinect 與 AR. Drone 2.0。

接上 Processing，與 Machine Learning 的 training software，整合出一套系統能透過 Kinect 辨識手勢，達到操控無人飛行器的效果。

NTU Gossip

I Built a webpage enabling people to create posts anonymously to a Facebook fanpage with more than 1,800 likes.

個人 Project，大三時因為 XX台大系列粉絲專頁突然興起，鑑於每個系列使用的匿名留言系統，都是同一個大陸公司寫的，大家不是很信任。於是我心血來潮某天下午用 Node.js 加上 MongoDB(後來改成 Parse)，從無到有自建一個匿名留言板。連點圖獲取驗證碼都自己寫，大約花了兩天時間。日後接上 Facebook Graph API 自動發文。為的是讓大家有個安心的匿名平台，甚至計畫開源給大家使用。

Gogobit

A powerful bitcoin assistant iOS app and its landing page.

iRelatives

I Designed and developed an iOS app to help dementia patients for their rehabilitation.

Course Project，系上某年開設的行動裝置程式設計期末專案，那時三人一組做出了一個給輕度或中度的失智老人的復健小遊戲。iRelatives 可以讓失智老人的親友或是他自己，建立自己的親友與小孩，可錄製聲音、拍攝照片以及建立稱謂和姓名。建立完之後，可以選擇三種遊戲模式，第一種是 Name Mode，即看照片猜名字或看名字猜照片。

第二種是聽聲音選照片或名字，第三種為前兩種混合，若滿分會予以鼓勵，低分也是會予以鼓勵再加油。希望藉由這款 App，幫助阿茲海默症的老人能延緩病情，甚至復健記憶。
